About The Class

This class was the next level in web development. During the class, I learned to design and develop creative websites and advanced my understanding of the primary frontend web languages HTML5, CSS, and JavaScript. The course utilized higher-level development tools and open source frameworks to achieve a modern work-flow. The class was an experimental sandbox that played with bleeding-edge web technologies with subversion and vigor!

Bash Me

A one page html document using ONLY the terminal (no GUI). First use of BASH. The html markup was coded in the terminal using NANO.

Love Letters

This project created three html5 semantic pages expressing my undying love AND instructions on how to use BASH, GIT, and GITHUB. The website shares my fondness for these three developer tools by creating a mashup standard operating procedure/love letter for each of them. Each page is uniquely styled with love and css. One of them is a break-up letter. Git commit and push to github were used obsessively.

Exquisite Corpse

A method invented by the surrealists by which a collection of words or images is collectively assembled. Each collaborator adds to a composition in sequence. This project constructed a creative narrative where each team member contributes a paragraph based on of what the previous person contributed. The design of the webpage is also a collaborative effort.

CSS Zen Garden

A starting template that combines html5 boilerplate with a sass/compass directory setup.

Sprite Sheet

A compass project and a themed web page that features 10 custom sprite icons (with rollovers). Compass' sprite generator was used to create a sprite sheet with at least 10 custom image icons with corresponding rollovers (20 images total) that are the same width and height and optimized as transparent png-24.

Animation

A compass project and a themed web page that features sprite animation. The images were created to line up like a flipbook. Animated using css animation

Message

A one page website using css grids and google analytics. The content for the project is a discrete unit of communication intended for consumption by a group of recipients or simply put - a message. Promoted the message with social networks and email blasts and then used google analytics to track how far the message reached and what technologies were used to consume it.

Faux Company

Used the modern workflow - git, github, html5 boilerplate, sass, compass, css grids, and google analytics to create a professional responsive website that sells something (product, fantasy, idea). Included custom icons using a compass generated sprite sheet. Design images were optimized and original. Git commit and push to github vigorously

Parallax Narrative

A narrative using the mechanic of css parallax scrolling with the theme "A Day in the Life..."

Wordpress

A self-hosted custom wordpress site from scratch using the underscores (_s) starter theme. The site should serve as a personal (professional) space to display your creative body of work (portfolio), as well as a space to document your current and future projects.
